#qpycalc#

qpycalc is a quantum gate calculator

It supports the following 1 qubit gates:
X
Y
Z
S
T
H

It supports input in Dirac's notation and these standard states
|0>
|1>
|+>
|->
|i>
|-i>
and their superposition


qp>|0> X
r: 1., phi: 0., theta: 0.0
r: 1.0, phi: 0.0, theta: 180.0
